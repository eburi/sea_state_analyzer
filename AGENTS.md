# AGENTS.md — compatibility shim

The canonical agent-agnostic briefing for this repository lives in [`AGENT.md`](AGENT.md).

If your tool auto-loads `AGENTS.md`, treat `AGENT.md` as authoritative and then continue with:

1. [`harness/README.md`](harness/README.md)
2. [`harness/conventions.md`](harness/conventions.md)
3. the latest report under [`docs/reviews/`](docs/reviews/)

This shim intentionally avoids duplicating rules so the repo has one canonical briefing source.
